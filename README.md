# DuoDirecte

DuoDirecte is a gamified school management mobile application designed to enhance the educational experience for students and teachers. Inspired by platforms like Duolingo, Pronote, and École Directe, DuoDirecte integrates various features to streamline school management while making it engaging and fun.

## Features

- **User Login**: Secure authentication for users to access their accounts.
- **Automatic Synchronization**: Seamless integration with existing school systems to synchronize grades and assignments.
- **Customizable Interface**: Users can personalize their app experience to suit their preferences.
- **Gamified Screen Time System**: Encourages productive use of time through gamification techniques.
- **Lesson Completion**: Similar to Duolingo, users can complete lessons and track their progress.
- **XP and League Systems**: Users earn experience points (XP) and can compete in leagues to enhance motivation.
- **Secure Messaging**: Facilitates communication between users in a safe environment.
- **Intelligent Notifications**: Smart alerts for assignments, grades, and other important updates.

## Installation

To get started with DuoDirecte, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/DuoDirecte.git
   ```

2. Navigate to the project directory:
   ```
   cd DuoDirecte
   ```

3. Install the dependencies:
   ```
   npm install
   ```

4. Run the application:
   ```
   npm start
   ```
